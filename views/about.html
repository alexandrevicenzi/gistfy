About

Gistify is a simple way to create snippets to share your code with others.

What this isn't

This isn't a pastebin like tool. To use Gistfy you need to have your code somewhere.

Supported Services

  • GitHub (public repository only)
  • GitHub Gist (public gist only)
  • Bitbucket (public repository only)

Code

The source code is available on GitHub.
